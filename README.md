âœ¨Key Notes for Users
True/False in CSV Format 2:
Use 1 for true, 0 for false in the correct_answer column.
Fill-in-the-Blank:
For CSV: Put the answer directly in correct_answer.
For JSON: Use "type": "fill_in_blank".
